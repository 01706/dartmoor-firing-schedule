# Dartmoor Hiking and Camping Information Map

Map which show the Military of Defence (MOD) training areas on Dartmoor, with a selectable date highlighting scheduled live firings.

Also included on the map are the permitted camping areas, access land (right to roam) and ground nesting bird sites which should be avoided during nesting season.


Visit the map at [dartmoor.andrewtrevena.co.uk](https://dartmoor.andrewtrevena.co.uk)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
